# tw-music-app

Music App built with TailwindCSS and AlpineJS

[Preview](https://indrasukma.github.io/tw-music-app/)

## Installation

```
git clone https://github.com/IndraSukma/tw-music-app.git
cd tw-music-app
npm install
```

## Usage

### Start local development

```
npm run watch
```

### Build production files

```
npm run minify
```
